🍎 Fruit Search Application (React)

A simple React-based web application that allows users to search and filter a list of fruits dynamically. The app updates results in real time as the user types, demonstrating core front-end development concepts.

🚀 Features
Real-time fruit search functionality
Dynamic filtering based on user input
Displays matching results instantly
Shows "No results found" when no matches exist
Responsive and simple user interface
🛠️ Technologies Used
React (useState, useEffect)
JavaScript (ES6+)
HTML5
CSS3

│── index.html

⚙️ How It Works
A predefined list of fruits is stored in the application
As the user types in the search input:
The input value is captured using React state
The list is filtered using JavaScript .filter()
Matching results are displayed dynamically
If no match is found, a message is shown
📦 Installation & Setup
Option 1: Run in Browser
Clone the repository:
git clone https://github.com/Yasser514/Build-a-Fruit-Search-App
Open index.html in your browser
Option 2: Run with React (Recommended)
Create a React app:
npx create-react-app fruit-search-app
cd fruit-search-app
npm start
Move the logic into src/App.js
